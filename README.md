# 📊 Challenge Alura Store - Análise de Dados com Python

Este projeto faz parte do **Challenge ONE Data Science**, uma iniciativa da **Oracle** em parceria com a **Alura Latam**. O objetivo é aplicar conceitos fundamentais de análise de dados utilizando bibliotecas nativas do Python para transformar dados brutos em insights estratégicos.

---

## 🎯 Propósito da Análise
A análise foi desenvolvida para a **Alura Store**, focando em compreender o desempenho de vendas, satisfação dos clientes e custos logísticos. O projeto simula um cenário real de tomada de decisão baseada em dados, integrando lógica de programação e visão de negócio.

## 🛠️ Tecnologias e Conceitos Aplicados
Para a construção deste projeto, foram utilizados:
* **Python**: Linguagem principal para o desenvolvimento da lógica.
* **Pandas**: Manipulação e extração de dados de arquivos CSV hospedados no GitHub.
* **Matplotlib**: Geração de visualizações gráficas para facilitar a interpretação dos resultados.
* **Lógica de Programação**: Uso de variáveis, listas, loops (`for`) e condicionais (`if/else`) para estruturar os insights.
* **Google Colab**: Ambiente de desenvolvimento em nuvem utilizado para a codificação.
* **Git & GitHub**: Ferramentas essenciais para versionamento e documentação profissional do projeto.

## 📈 Insights e Resultados Obtidos
Através do processamento dos dados, alcançamos os seguintes marcos:
1. **Faturamento Total**: A Alura Store atingiu um faturamento bruto de **R$ 1.534.509,12**.
2. **Vendas por Categoria**: A categoria de **Eletrônicos** consolidou-se como a principal fonte de receita da operação.
3. **Satisfação do Cliente**: A média de avaliação foi de **3.98**. Implementamos uma lógica condicional que classifica essa satisfação como **mediana**, indicando oportunidades de melhoria nos processos de atendimento ou produto.
4. **Desempenho de Produtos**: O **Micro-ondas** foi o item campeão de vendas (60 unidades), enquanto o **Headset** apresentou a menor saída (33 unidades).
5. **Logística (Frete)**: Identificamos que o estado do **Acre (AC)** possui o maior custo de frete médio, o que sugere uma necessidade de revisão da malha logística para a região Norte.

## 📁 Estrutura do Projeto
* `AluraStoreBrasil.ipynb`: Notebook contendo todo o código de extração, tratamento e visualização de dados.

## 🚀 Como executar o projeto
1. Acesse este repositório e baixe o arquivo `.ipynb`.
2. Faça o upload do arquivo no [Google Colab](https://colab.research.google.com/).
3. Execute as células em ordem, iniciando pela seção **Importação dos dados** para carregar as bases externas via URL.
