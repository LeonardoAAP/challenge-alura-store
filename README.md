📊 Challenge Alura Store - Análise de Dados com Python
Este projeto faz parte do Challenge ONE Data Science, uma iniciativa da Oracle em parceria com a Alura Latam. O objetivo é aplicar conceitos fundamentais de análise de dados utilizando bibliotecas nativas do Python para transformar dados brutos em insights estratégicos.

🎯 Propósito da Análise
A análise foi desenvolvida para a Alura Store, focando em compreender o desempenho de vendas, satisfação dos clientes e custos logísticos. O projeto simula um cenário real de tomada de decisão baseada em dados.

🛠️ Tecnologias e Conceitos Aplicados
Para a construção deste projeto, foram utilizados:

Python: Linguagem principal.

Pandas: Manipulação e extração de dados de arquivos CSV.

Matplotlib: Visualização de dados através de gráficos de barras e horizontais.

Lógica de Programação: Uso de variáveis, listas, loops (for) e condicionais (if/else) para estruturar os insights.

Google Colab: Ambiente de desenvolvimento em nuvem.

Git & GitHub: Versionamento e documentação do projeto.

📈 Insights e Resultados Obtidos
Durante o desenvolvimento, alcançamos os seguintes marcos:

Faturamento Total: A Alura Store atingiu um faturamento de R$ 1.534.509,12.

Vendas por Categoria: A categoria de Eletrônicos consolidou-se como a principal fonte de receita.

Satisfação do Cliente: A média de avaliação foi de 3.98. Implementamos uma lógica condicional que classifica essa satisfação como mediana, indicando necessidade de melhorias nos processos.

Desempenho de Produtos: O Micro-ondas foi o item mais vendido (60 unidades), enquanto o Headset teve a menor saída (33 unidades).

Logística (Frete): Identificamos que o estado do Acre (AC) possui o maior custo de frete médio, sugerindo uma oportunidade para revisão da malha logística.

📁 Estrutura do Projeto
AluraStoreBrasil.ipynb: Notebook contendo todo o código de extração, tratamento e visualização.

README.md: Documentação completa do projeto.

.gitignore: Arquivo para evitar o versionamento de arquivos desnecessários do ambiente Python.

🚀 Como executar o projeto
Acesse o repositório e baixe o arquivo .ipynb.

Faça o upload do arquivo no Google Colab.

Execute as células em ordem, começando pela Importação dos dados para carregar as bases via URL.
